Project Overview

This project presents a Hybrid IoT-LSTM Framework designed for real-time monitoring and prediction of CO₂ levels in confined indoor environments.

The system leverages a trained Long Short-Term Memory (LSTM) model to forecast air quality trends based on environmental conditions and occupancy levels. This enables proactive ventilation control and improved indoor air quality management.

Key Features

  • Real-time CO₂ prediction using AI
  • Dynamic ventilation control recommendations
  • Occupancy-aware air quality modeling
  • Interactive dashboard with live visualization
  • Dataset upload and analysis capability
  • Simulated IoT environment with live updates

Technology Stack

  • Machine Learning: LSTM (TensorFlow/Keras)
  • Backend: Flask (Python)
  • Frontend: HTML, CSS, JavaScript
  • Visualization: Chart.js

Applications

This system can be applied in:

  • Smart homes and offices
  • Classrooms and lecture halls
  • Healthcare environments
  • Industrial indoor spaces

Future Scope

  • Integration with real IoT sensors
  • Automated HVAC system control
  • Multi-room monitoring
  • Cloud-based deployment
